Rounded Corners

 .img-rounded class 는 이미지에 둥근 모서리(rounded corner)를 추가한다. (IE8 do not support rounded corners):

Example

<img src="cinqueterre.jpg" class="rounded" alt="Cinque Terre" width="304" height="236">
Try it Yourself »

Circle

 .rounded-circle class 는 이미지 모양을 원(circle)으로 만들어준다. (IE8 do not support rounded corners):

Example

<img src="cinqueterre.jpg" class="rounded-circle" alt="Cinque Terre" width="304" height="236">
Try it Yourself »

Thumbnail

 .img-thumbnail class 는 이미지 모양을 축소판(thumbnail)으 로 만들어준다.:

Example

<img src="cinqueterre.jpg" class="img-thumbnail" alt="Cinque Terre" width="304" height="236">
Try it Yourself »

Aligning Images

.float-right 클래스를 사용하여 이미지를 오른쪽으로, .float-left를 사용하여 왼쪽으로 이미지를 플로팅합니다.:

Example

<img src="paris.jpg" class="float-left">
<img src="paris.jpg" class="float-right">
Try it Yourself »

Centered Image

유틸리티 클래스 .mx-auto(margin:auto) 및 .d-block(display:block)을 이미지에 추가하여 이미지를 중앙에 배치합니다.:

Example

<img src="paris.jpg" class="mx-auto d-block">
Try it Yourself »

Responsive Images

이미지는 모든 크기로 제공됩니다. 스크린도 마찬가지입니다. 반응형 이미지는 화면 크기에 맞게 자동으로 조정됩니다.

<img> 태그에 .img-fluid 클래스를 추가하여 반응형 이미지를 만듭니다. 그러면 이미지가 부모 요소에 맞게 잘 조정됩니다.

.img-fluid 클래스는 이미지에 max-width: 100%; 와 height: auto; 를 적용합니다. : :

Example

<img class="img-fluid" src="img_chania.jpg" alt="Chania">
